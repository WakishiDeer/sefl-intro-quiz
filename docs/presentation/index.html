AI × リアルタイム Web アプリ  |  v0.1.0  |  2026-03-24

OVERVIEW

Self-Intro Quiz とは？

参加者の自己紹介情報をもとに AI が4択クイズを自動生成 するリアルタイム Web アプリです。

社内懇親会やアイスブレイクで、初対面でも盛り上がれる体験を提供します。

🤖

AI 自動生成

参加者のプロフィールから Azure OpenAI が面白いクイズを自動生成。準備不要！

⚡

リアルタイム

Socket.IO による全端末同期。回答・スコア・タイマーがリアルタイムに反映されます。

📱

ブラウザだけ

インストール不要。PC でもスマホでもブラウザからすぐ参加できます。

🎭

テーマ切替

5種類のテーマ（subtle / fun / party / cyber / sakura）でクイズの雰囲気をカスタマイズ。

STEP 1

🏠 トップページ

シンプルなトップ画面。「ルームを作成する」または「ルームに参加する」から始められます。

STEP 2

✨ ルーム作成 & 参加

ルーム作成（Host）

ニックネームを入力するだけでルームが作成されます。6文字のルームコードが自動生成されます。

ルーム参加

ルームコードを入力するか、開催中のルーム一覧から選んで参加できます。招待リンクにも対応。

STEP 3

👥 ロビー — 参加者が集まる

参加者のオンライン状態がリアルタイムに表示。Host は参加者管理やテーマ選択ができます。

3人が参加したロビー画面（Host 視点）

STEP 4

📝 プロフィールを入力

出身地・趣味・特技・好きな食べ物・意外な事実・自由記述 — 6つのフィールドに自己紹介を入力します。
この情報をもとに AI がクイズを生成します。

THEMES

🎨 5つのテーマで雰囲気を演出

ルームの雰囲気に合わせてテーマを切り替え。色・アニメーション・エフェクトが一括で変わります。

スレート / インディゴ基調。落ち着いた雰囲気のデフォルトテーマ。

アンバー / オレンジ基調。ポップで明るい雰囲気。

ダークシアン / パープル基調。近未来的なダークモード。

フクシア / ピンク / バイオレット基調。華やかなパーティ向け。

ピンク / ローズ基調。桜をモチーフにしたテーマ。

📦 モノレポ構成

  • packages/shared — 型定義・定数・バリデーションスキーマ（server/client 共通）
  • packages/server — Express + Socket.IO サーバ（DDD: Room Context + Quiz Context）
  • packages/client — React SPA（ページ・コンポーネント・テーマ）

🧩 ドメインモデル（Bounded Context）

  • Room Context — ルーム作成・参加者管理・Host 権限・フェーズ遷移
  • Quiz Context — クイズ生成・出題進行・回答受付・スコア計算
  • 2つのコンテキストは roomCode で疎結合に連携
FEATURES

✨ 主な機能・こだわりポイント

🔄

リアルタイム同期

Socket.IO による WebSocket 通信。参加者の入退室・回答状況・スコアがリアルタイムに同期。

🚪

途中参加サポート

クイズ進行中でもルームに参加可能。次の問題から回答に参加でき、スコアは参加問数ベースで計算。

⏱️

タイマー同期

サーバの timerEndsAt を基準に、全端末で正確にカウントダウン。requestAnimationFrame ベース。

🎨

5種類のテーマ

subtle / fun / party / cyber / sakura。全33色 + 10種アニメーション variants で統一管理。

✉️

ルーム招待

他のルームへ招待を送信。フローティング通知でワンクリック参加が可能。

😄

リアクション

👏❤️😂🎉 などのリアクションを送信。パーティクルエフェクトで画面が盛り上がる！

🎤

インタビュー機能

「気になる👀」投票50%以上で自動遷移。クイズの答えについて本人に話を聞けるフェーズ。

🛡️

セキュリティ

入力サニタイズ、レート制限、Zod バリデーション、XSS 対策を全層で実施。
